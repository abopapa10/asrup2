# Bursa Psikolog — Ahmet Alparslan Sancar

Premium local SEO landing page for **bursapsikologsancar.com**.

## Stack

- Next.js 15 (App Router)
- Tailwind CSS
- Framer Motion
- Lucide React

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production

```bash
npm run build
npm start
```
